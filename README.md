# BrowserHistory
利用Java导出Chrome的浏览历史
# 运行方式
克隆仓库，关闭Chrome浏览器，点击运行ExportChrome.bat，在项目路径下导出Chrome浏览历史的Excel文件
# 支持环境
windows10
